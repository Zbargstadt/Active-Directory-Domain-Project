# Active-Directory-Domain-Project ( IN PROGRESS )
This is the documentation of my Active Directory Home lab, designed to mimic a real-world enterprise environment. This will be hosted on my own PC using VMware workstation, the environemnt will include both a server, a client pc, and a ticketing system. Therefore once this project is finished I can hopefully practice some real world tickets that one would receive in a help desk role.
